HEALTHCARE TECHNOLOGIST.

There are serious problems with Healthcare in the U.S., many of which I believe would be rapidly improved if we had fluid, interoperable data and powerful tools to use it.

I believe the patient should be the conduit through which all this data flows, and I've helped drive this concept with my work on "Blue Button for America" as a Presidential Innovation Fellow, writing code to liberate XML medical records directly to Veterans.

I'm also a proponent of open-source. The federal government has invested billions in building powerful systems which run the infrastructure of the largest healthcare provider in the US, the VA. I'm helping them give it away freely to the everyone.
